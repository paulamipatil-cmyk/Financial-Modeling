# Financial-Modeling

Three-Statement Financial Model — CFA Level I Practical Skills Module
Blu_Containers_Worksheet_Beginner.xlsx
Completed as part of the mandatory CFA Level I Practical Skills Module (PSM) — a guided, hands-on exercise in building an integrated financial model. Covers Income Statement, Balance Sheet, and Cash Flow Statement with scenario analysis across pricing assumptions, working capital modeling, and EBITDA margin analysis.
