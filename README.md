# Financial-Modeling

Completed as part of the mandatory CFA Level I Practical Skills Module (PSM) — a guided, hands-on exercise in building an integrated financial model for a container pricing case study. The Excel workbook contains five structured tabs (Cover, Summary, Assumptions, Scenarios, Model) covering a full Income Statement, Balance Sheet, and Cash Flow Statement. The PDF provides a rendered view of the model for quick reference without downloading the spreadsheet.

Key techniques: Three-statement modeling · Scenario analysis · EBITDA margin analysis · Working capital modeling · Balance sheet reconciliation
